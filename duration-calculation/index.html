Duration Calculator

Computes effective ticket duration with overlap handling, banking hours, weekends & public holiday deduction

Input File
Expected file structure:
  • Data sheet — two header rows; data from row 3 onwards. Required columns: ATM ID, State, Start Date, Start Time, End Date, End Time, Duration, Post banking Hours, Weekend Holiday Hours, Public Holiday Hours, Actual Downtime, Total Downtime
  • Calender sheet — columns: DATE, STATE, HOLIDAY (Weekend Holiday entries apply to all states; other entries are state-specific)
  • Overlap rule: if a ticket for the same ATM starts before the previous ticket closes, its effective duration = (prev end − prev start) + (curr end − prev end) = curr end − first start
